# 深度Review: Self-Augmenting Retrieval for Diffusion Language Models

## 1. 核心方法与贡献

本论文提出SARDI框架，利用离散扩散语言模型（DLMs）的中间解噪状态中的"低置信度"预测作为检索信号，实现动态的检索增强生成。核心创新在于分离检索阈值τ_q和提交阈值τ_c，允许投机性的未来预测在足够稳定前就用于指导检索，从而在多跳问答中更早地识别桥接实体。该方法无需训练，与任何离散扩散模型兼容，在五个QA基准上以8倍更高的吞吐量超越现有方法。

## 2. 技术细节

**核心机制：置信度分离**
- 在每个解噪步骤t，模型为所有位置i预测置信度 $c_i = \max_{v \in V} p_\theta(v|x^t, q, D^t)$
- 构造代理序列：$\tilde{x}^t_i = \begin{cases} x^t_i & \text{if } x^t_i \neq \text{[MASK]} \\ \arg\max_v p_\theta(v|x^t, q, D^t) & \text{if } c_i \geq \tau_q \\ \text{[MASK]} & \text{otherwise} \end{cases}$
- 检索查询：$s_t = q \parallel \text{Detokenize}(\tilde{x}^t)$，融合问题和中间响应
- 证据更新：$D^{t-1} \leftarrow R(s_t, K)$，动态替换上下文

**并行解码的理论基础**：论文论证RAG通过使输出令牌受限于检索证据D，大幅降低令牌间的互信息，使其更接近条件独立，从而支撑高效的并行采样。

## 3. 优点与局限

### 优点
1. **创意设计**：将扩散轨迹作为检索的前向看入（lookahead）这一想法精妙而自然，充分利用了DLMs的并行结构优势，相比AR模型的FLARE方法更稳健（并行预测无误差累积）
2. **无训练部署**：完全即插即用，无需任何微调或强化学习，降低了工程复杂度和计算成本，实用性强
3. **充分的实证验证**：通过5个多跳QA基准验证，并在5.5节提供了RAG下令牌依赖性降低的直接证据，理论与实验结合紧密

### 局限
1. **方法通用性受限**：虽声称"任何离散扩散模型"均适用，但框架隐含依赖于模型能输出可靠的位置级置信度分数。对于置信度校准不良的模型，分离阈值τ_q和τ_c的有效性难以保证；且仅在多跳QA领域实验，未探索在其他任务（如长文本生成、代码生成）的泛化性
2. **检索开销分析不足**：每步都执行检索（方程5）将显著增加总体延迟。虽声称"8倍吞吐量"，但对比缺乏细节：是否包括检索时间？与单步检索相比的实际壁钟时间是多少？动态检索带来的额外计算成本被掩盖
3. **查询构造的原始性**：简单的串联拼接（$q \parallel r_t$）可能不是最优方案，缺乏对查询质量的深度分析。τ_q=0的默认选择虽在5.4有消融研究，但未探索更复杂的查询融合策略（如注意力加权、重要性排序）

# Plug-and-Play Guidance for Discrete Diffusion Models深度Review

## 1. 核心方法与贡献

本论文提出**GILC（Gradient-Informed Logit Correction）**，一个即插即用的训练免费框架，用于离散扩散模型的可控生成。核心创新在于：（1）利用预训练的去噪网络作为变分代理估计值函数梯度；（2）引入Jacobian-free机制直接修正干净预测的logits，规避离散空间中的梯度不稳定性问题；（3）建立与策略梯度的形式联系，支持可微和不可微的奖励函数。该方法无需微调即可在DNA、蛋白质和分子生成任务中达到SOTA性能。

## 2. 技术细节

### 2.1 问题建模
论文将可控生成表述为优化目标（Eq. 5）：
$$p_\theta^r(x) = \arg\max_q \mathbb{E}_{x\sim q}[r(x)] - \beta D_{KL}(q(x)||p_\theta^{pre}(x))$$
其闭形式解为：$p_\theta^r(x) \propto p_\theta^{pre}(x)\exp(r(x)/\beta)$

### 2.2 核心机制
**最优反向过程**：将指导生成转化为计算值函数梯度
$$p_\theta^r(z_s|z_t) = \frac{p_\theta(z_s|z_t)\exp(v(z_s)/\beta)}{\sum_{z_s}p_\theta(z_s|z_t)\exp(v(z_s)/\beta)}$$

其中软值函数定义为：$v(z_s) = \beta\log\mathbb{E}_{p_\theta(x|z_s)}[\exp(r(x)/\beta)]$，在实践中近似为期望奖励。

**Logit修正策略**（关键创新）：
- 采用Gumbel-Softmax技巧与Straight-Through估计器维持离散空间的梯度流
- **关键insight**：直接在干净预测logits上应用奖励梯度修正，而非计算完整的Jacobian：
$$x_\theta^r = x_\theta + \lambda \cdot \nabla_{x_\theta} v(x_\theta)$$
这规避了在高维离散空间中Jacobian矩阵导致的数值不稳定性。

### 2.3 离散扩散框架
采用Sahoo et al.的简化框架，前向过程直接在干净数据$x$与先验分布间插值：
$$q(z_t|x) = \text{Cat}(z_t; \alpha_t x + (1-\alpha_t)\pi)$$
使用吸收态（MASK token）作为先验，使后验分布（Eq. 3）具有可处理的形式。

## 3. 优点与局限

### 优点
1. **训练免费与即插即用特性**：无需微调或额外训练，可直接使用现有预训练模型和奖励函数，显著降低计算成本和部署复杂度。

2. **理论完善性与实用性结合**：从值函数梯度的Taylor展开出发建立理论基础，同时通过Jacobian-free logit修正实现稳定有效的梯度计算，优雅地解决离散空间的梯度不稳定问题。

3. **广泛的应用适用性**：支持可微与不可微奖励函数（通过策略梯度建立联系），涵盖DNA、蛋白质、分子设计等多个科学领域，实验结果表明性能超越现有训练免费方法且可比肩微调方法。

### 局限
1. **价值函数近似的精度问题**：将软值函数$v(z_s) = \beta\log\mathbb{E}[e^{r(x)/\beta}]$近似为期望奖励$\mathbb{E}[r(x)]$（在小$\beta$时有效），但该近似在中等$\beta$值下可能引入偏差，论文未充分分析近似误差对引导效果的影响。

2. **可扩展性与计算效率权衡**：虽然相比SMC和重要性采样方法更高效，但仍需在每个扩散步骤计算奖励梯度，对极长序列或实时应用可能存在延迟。论文缺乏关于序列长度、词汇表大小等因素对计算复杂度的详细分析。

3. **奖励信号设计依赖性高**：方法的性能高度依赖奖励函数的质量和设计，但论文对如何在实践中构造有效的奖励函数指导不足，特别是对于非凸或多模态奖励景观的处理缺乏讨论。

# NAVIRA论文深度Review

## 1. 核心方法与贡献

本论文提出NAVIRA，一种针对掩码扩散语言模型(MDMs)的推理时解码策略，通过**解耦(decoupling)token质量评分和生成过程**以及**引入温度控制的随机重掩码机制**来改进文本生成质量。相比前作PRISM采用单次前向传播同时完成质量评分和重生成的耦合方案，NAVIRA采用两次前向传播将这两个操作分离，使得被标记为低质量的token不再污染其自身的修正上下文，同时通过随机采样而非贪心选择来平衡流畅性与多样性。

## 2. 技术细节

### 核心创新点：

**2.1 解耦的两阶段推理机制**

第一阶段：质量评分与重掩码选择
$$T_t \subseteq C_t, \quad T_t = \text{Sample}(\pi_i(\tau_t)), \quad \pi_i(\tau_t) \propto \exp(-s_i/\tau_t)$$

其中$g_\phi(x^{(t)})_i$为token $i$的质量分数，$\tau_t$为温度参数。

第二阶段：从清洁上下文重生成
$$\tilde{x}^{(t)} = x^{(t)} \text{ with positions } T_t \text{ masked}$$
$$x^{(t+1)} \sim f_\theta(\cdot|\tilde{x}^{(t)})$$

**对比PRISM(耦合方案)**：
- **PRISM**：单次前向传播 $f_\theta(\cdot|x^{(t)}), g_\phi(x^{(t)})$ → 在包含错误token的原始上下文中生成修正

- **NAVIRA**：
  - 第一次前向：评分 $g_\phi(x^{(t)})$ 
  - 第二次前向：从$\tilde{x}^{(t)}$（已掩码错误位置）重生成，避免错误token的条件影响

**2.2 温度控制的随机重掩码**

相比PRISM的确定性top-K规则，NAVIRA采用：
- **固定温度**$\tau$：始终保持相同的随机性程度
- **调度温度**：$\tau_t$随迭代$t$变化，常见策略为线性衰减或指数衰减，早期阶段更激进地探索，后期聚焦纠正

这种设计减少了对同一低分位置的重复选择，缓解了模式坍缩现象。

### 算法框架（改进后）：

```
Algorithm: NAVIRA Inference Step
Input: Current sequence x^(t), quality head g_φ, backbone f_θ, temperature τ_t
1. First forward pass: compute quality scores
   s_i = g_φ(x^(t))_i  for all i ∈ C_t (clean positions)
2. Stochastic remasking: sample positions from temperature-controlled distribution
   π_i(τ_t) ∝ exp(-s_i/τ_t)
   T_t ~ Sample(π(τ_t))  // sample K positions without replacement
3. Create cleaned context
   x̃^(t) = x^(t) with positions T_t masked
4. Second forward pass: regenerate from cleaned context
   ŷ_i ~ f_θ(·|x̃^(t))_i  for all masked positions
5. Update: x^(t+1) = {ŷ_i for unmasked positions, x^(t)_i for remasked positions}
```

**关键公式解释**：
- 质量分数映射到重掩码概率：$\pi_i(\tau_t) = \frac{\exp(-s_i/\tau_t)}{\sum_j \exp(-s_j/\tau_t)}$
- 低分数token（$s_i$小）→ 高重掩码概率（分子大）
- 温度$\tau_t$增大 → 分布更均匀，探索性更强；$\tau_t$减小 → 聚焦低分位置

## 3. 优点与局限

### 优点：

1. **问题诊断精准且解决方案简洁**：清晰识别了PRISM中的"耦合问题"（erroneous tokens still condition regeneration），提出的两阶段解耦方案设计优雅，易于实现且无需修改预训练模型。

2. **多维度实验验证与可视化**：
   - 受控消融实验在**相同前向传播预算**条件下对比（关键细节），证明改进来自策略而非计算量增加
   - 同时评估多个维度：困惑度(PPL)、熵(entropy)、LLM-judge评分，避免单一指标的局限
   - Figure 1的流程图清晰展示了两方案的区别

3. **平衡探索与利用的理论动机**：引入温度控制的随机采样而非贪心选择，从信息论角度缓解"反复修正相同位置导致多样性丧失"的问题，有文献支撑（Holtzman et al., 2020关于确定性解码的危害）。

### 局限：
